Сайт - место в интернете для размещения какой-либо информации, которая
выкладывается владельцем для общего обозрения. Сайты могут состоять из
множества связанных между собой страниц. Сайты бывают различных типов,
это может быть блог, интернет портал, тематический сайт и многое другое.
Но на деле сайт это зарегистрированное на одного или нескольких пользователей
место в интернете, состоящие из различных HTML, CSS, PHP, Java Script файлов.
